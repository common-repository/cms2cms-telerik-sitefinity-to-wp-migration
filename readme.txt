Plugin for automated data migration from Sitefinity to WordPress. Convert Sitefinity to WordPress easily and fast.

== Description ==

If you wish to switch from your current Telerik Sitefinity and WordPress sounds like a perfect choice for you – CMS2CMS Migration Plugin will help you to go through the migration path smoothly and trouble-free. A full-scale migration takes up from 15 min (depending on your website size) and doesn’t expect any special skills to complete it.

= HOW THIS PLUGIN HELPS: =
- Ease of Use – CMS2CMS is tailor made for each client, that is suitable for both web developers and average users to work with;
- High Speed – CMS2CMS migration takes approximately 15 min to complete the conversion (though it may depend on a website size);
- Free Demo Available – CMS2CMS provides its users with a possibility to see the website migration process in action and to estimate the results;
- Content Mapping - choose what data items to move to your New website;
- Complete Migration Price - depends on how many content items you want to transfer to WordPress.


= WHAT WILL BE MOVED FROM SITEFINITY TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.


= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing Sitefinity site to the new WordPress website.


= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated Sitefinity To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
